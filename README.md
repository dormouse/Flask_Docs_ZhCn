## 制作本地文档

### Html 文档

主要步骤为：

* 克隆本项目
* 创建虚拟环境（示例使用 anaconda ）
* 安装依赖
* 生成文档（文档生成在 _bulid 目录下)

命令示例：

```shell
git clone https://github.com/dormouse/Flask_Docs_ZhCn.git
cd Flask_Docs_ZhCn/
conda create -n flask_doc
source activate flask_doc
pip install -r requirements.txt
make html
```
